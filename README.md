# CRUD Users
> Short introduction about the App.

- A Java web application based on the Spring framework was developed in which we implemented the User module with CRUD (Create, Retrieve, Update & Delete) operations.


## Technologies: 
* Spring Boot Web
* Spring Data JPA & Hibernate
* MySQL Database
* Thymeleaf
* HTML5 & Bootstrap
* JUnit 5 & AssertJ
* Spring Data JPA Test.

## Software programs: 
* Java Development Kit (OpenJDK)
* IntelliJ IDEA Ultimate
* MySQL Community Server
* MySQL Workbench
